# Projeto-Dashboard-de-Vendas-com-Python01
nalisei +9.000 pedidos de uma loja de varejo usando Python. O processo foi dividido em 4 etapas:  1️⃣ Coleta e leitura dos dados com pd.read_csv() 2️⃣ Limpeza e preparação com Pandas — conversão de datas, tratamento de nulos e criação de colunas auxiliares 3️⃣ Agregação com groupby() para responder perguntas de negócio: faturamento por categoria, região e sub-categoria 4️⃣ Visualização com Matplotlib e Seaborn — 6 gráficos reunidos em um dashboard exportado em PNG  Insight encontrado: pedidos com desconto acima de 30% operam consistentemente no prejuízo, independente da categoria.  Ferramentas: Python · Pandas · Matplotlib · Seaborn Dataset: Superstore (Kaggle)  #Python #DataAnalysis #DataVisualization #Pandas #Portfolio
