# Dashboard de Vendas com Python

Análise de +9.000 pedidos de uma loja de varejo americana usando Python, com foco em responder perguntas de negócio e visualizar os resultados em um dashboard.

---

## Objetivo

Explorar o dataset Superstore (Kaggle) para identificar padrões de faturamento, lucro e comportamento de vendas por categoria, sub-categoria e região.

---

## Etapas do Projeto

1. **Coleta** — leitura dos dados com `pd.read_csv()`
2. **Limpeza** — conversão de datas, tratamento de nulos e criação de colunas auxiliares
3. **Agregação** — uso de `groupby()` para responder perguntas de negócio
4. **Visualização** — 6 gráficos reunidos em um dashboard exportado em PNG

---

## Principal Insight

> Pedidos com desconto acima de 30% operam consistentemente no prejuízo, independente da categoria.
